PokeSync (Codename: Paragon)

A discontinued Java 2D MMORPG project based around the famous Pokémon franchise.

PokeSync is a 2D Pokémon MMORPG that was aimed for public alpha release in February 2016. The project had been in development for almost a year. The game consisted of client and server side software both written from scratch using Java.

I am the founder of the project and was the lead developer of the team. The development team consisted of five people contributing to game design, graphics, website and the actual game. However, unfortunately due to a lack of interest within the development team, the project was discontinued by December 2015.

A list of people I have worked with on this project:

  • Graphics Designer Cris Kevin Bjørndal
  • Web Developer Collin Haines
  • Game Designer Cameron Couch
  • Assistant Game Design Rachel Couch
